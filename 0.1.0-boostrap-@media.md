# Bootstrap

#### web Responsive

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * 响应式网页布局特点
 * 
 * 
 */

1) 网页宽度: 跟随用户屏幕自动跳这鞥

2) 元素大小: 基本均使用 相对大小

3) 字体:  rem / em ... 等

4) 布局: float / flex 

5) 媒体查询: 根据一个或多个设备类型、具体特点和环境来应用样式                   // 根据设备尺寸

```


#### @media

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * '@'                        // 在 html 中为一种规则 
 * 
 * 
 */

1) @chartset: 定义编码

2) @impot: 引入 CSS 文件( 模块化 )                 // 可以在当前 CSS 文件中引入其他 CSS文件

3) @font-face: 自定义字体

4) @keyframes: animation 中定义动画帧

5) @media: 媒体查询( 根据一个或多个设备类型、具体特点和环境来应用样式  )


//-------------------------------------------------------------------------------------------------------------------//


/**
 * 媒体设备
 * 
 * 
 */

1) all: 所有设备

2) print: 打印机设备

3) screen: 彩色的电脑屏幕

4) speech: 听觉设备( 针对有视力障碍的人士，可以把网页内容以语音的方式呈现 )

* tty, tv, projection, handeld, braille, embossed, aural 等几种类型已经在 媒体查询4 中废弃


//-------------------------------------------------------------------------------------------------------------------//


/**
 * 媒体特性
 * 
 * 
 */

1) width                                // 宽度

    min-width: ;
    
    max-width: ;
    
2) height                               // 高度

    min-height: ;

    max-height: ;

3) orientation                          // 方向

    landscape: ; ( 横屏 )
    
    partrait: ; ( 竖屏 )

4) aspect-ratio                         // 宽度比

5) -webkit-device-piexl-ratio-          // 像素比( -webkit- 内核私有属性 )


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
